# <p align="center">Jitsi 会议</p>

Jitsi 会议是一套开源项目，助力用户使用和部署视频会议平台，提供顶尖水准的音视频质量与丰富功能。

<hr />

<p align="center">
<img src="https://raw.githubusercontent.com/jitsi/jitsi-meet/master/readme-img1.png" width="900" />
</p>

<hr />

Jitsi 会议的核心功能包括（但不限于）：

* 支持所有主流浏览器
* 移动应用程序
* 用于集成的网页及原生软件开发工具包（SDK）
* 高清音视频
* 内容共享
* 举手发言与互动表情
* 支持私聊的聊天功能
* 会议投票
* 虚拟背景

以及更多功能！

## 使用 Jitsi 会议

Jitsi 会议基于浏览器运行，操作简便。访问 [meet.jit.si](https://meet.jit.si) 即可试用，该平台支持灵活扩展且完全免费。只需拥有谷歌、脸书或 GitHub 账号，即可发起会议，全浏览器兼容！

使用移动设备？无需担心！你可以通过手机浏览器访问，也可下载功能完备的官方移动应用：

## 搭建自有部署实例

若你希望搭建专属的 Jitsi 会议部署实例，请查阅 [官方手册](https://jitsi.github.io/handbook/docs/devops-guide/) 开启部署流程。

我们提供 Debian 系统安装包与完善的 Docker 配置方案，最大限度简化部署操作。高级用户也可选择从源代码编译所有组件。

最新版本信息可点击 [此处](https://jitsi.github.io/handbook/docs/releases) 查看。

## Jitsi 会议即服务（JaaS）

如果你既想要搭建自有实例的品牌定制能力，又不想耗费精力处理监控、扩容与版本更新等复杂运维工作，那么 JaaS 服务或许是你的理想之选。

[8x8 公司旗下 Jitsi 会议即服务（JaaS）](https://jaas.8x8.vc) 是一款企业级视频会议平台，能帮助开发者、各类组织及企业轻松构建并部署视频解决方案。借助 JaaS 服务，你可直接调用 Jitsi 的全部功能，且由我们的全球平台提供技术支撑，让你专注于打造安全合规、具备品牌特色的视频会议体验。
